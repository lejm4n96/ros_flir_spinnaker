## About The Project
The work is a ros driver for Gige-V FLIR thermal cameras supported by Spinnaker SDK

### Built With

* camera_info_manager
* dynamic_reconfigure
* image_proc
* cv_bridge
* image_transport
* nodelet
* rospy
* roscpp
* roslaunch
* sensor_msgs
* message_generation
